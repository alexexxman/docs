## 成功码

| 返回码   | 描述                       |
| ----- | ------------------------------ |
| 00000 | 成功码只有一个，其他所有返回码都是表示执行出现异常，是错误码 |

## Error Codes

| 返回码 | 描述             |
| ---------- | --------------------------------- |
| Axxxx      | 参数错误   |
| E0001      | 白名单列表未添加该ip值   |
| E0002      | 备注必须是20位字符以内组合 |
| E0003      | 项目不存在或项目状态异常 |
| E0004      | 项目未开启相关设置 |
| E0005      | 不支持的币种        |
| E0006      | 提币金额需大于0 |
| E0007      | 提币金额精度错误 |
| E0008      | 地址不合法                  |
| E0009      | 业务编号重复    |
| E0010      | 钱包不存在            |
| E0011      | 审批流程不存在  |
| E0012      | 审批流程已删除 |
| E0013      | 审批流程已关闭       |
| E0014      | 转出业务未查询到订单信息 |
| E0015      | 项目订单不存在     |
| E0018      | 订单编号不允许重复     |
| E0019      | 订单编号长度超过128位字符     |
| E0020      | 不支持的法币     |
| E0021      | 订单备注长度超过256位字符     |
| E0022      | 订单金额需大于0     |
| E0023      | 订单金额精度错误     |
| E0024      | 订单不存在     |
| E0025      | 付款人信息有误     |
| E0026      | 订单超时时间至少十分钟     |
| E0027     | 批量生成地址数量超出限制  |
| E0028     | 不支持的地址状态值        |
| E0029     | 至少一个修改项            |
| E0030     | 地址更新失败              |
| E0031     | 创建收银台失败            |
| E0032     | 无效的memo                |
